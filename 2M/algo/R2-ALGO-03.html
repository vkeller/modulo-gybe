Complexité (programme Python)#

Voici un programme Python :

n = 20  # Limite supérieure pour a, b, c

for a in range(1, n):
    for b in range(a, n):  # b >= a pour éviter les doublons
        for c in range(b, n):  # c >= b
            if a**2 + b**2 == c**2:
                print(a,b,c,"respectent la règle")
